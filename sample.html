> Using jQuery, you can use $(' ') to access elements in an HTML page. They are called query selectors.
To access a class you use "." and to access an id you use "#"
For example.. - To find "first_class" using jQuery we could write $('.first_class') - In Javascript, it would be document.getElementsByClassName('first_class') - To find "first_id" we would write $('#first_id') - In Javascript, it would be document.getElementById('first_id')
